# MoodLog AI

감정을 기록하고 회고 질문, 생각 재구성, 내일의 작은 행동을 제안하는 감정 일기 Agent입니다.

## 핵심 기능

- 오늘 있었던 일, 감정, 필요한 욕구, 반복 생각 입력
- 감정 강도, 에너지, 스트레스 점수를 바탕으로 감정 상태 분석
- 생각 재구성 문장과 지금 필요한 돌봄 행동 추천
- 회고 질문과 정리된 일기 초안 자동 생성
- 일기 저장 및 최근 감정 흐름 요약

## 실행 방법

1. `emotion-diary-agent/index.html`을 브라우저에서 엽니다.
2. 오늘 있었던 일과 감정 상태를 입력합니다.
3. `Agent 분석` 버튼을 누릅니다.
4. 생성된 일기 초안을 저장하거나 복사합니다.

## 과제 설명 요약

MoodLog AI는 감정을 말로 정리하기 어려운 문제를 해결하기 위한 개인 성장 Agent입니다. 사용자가 하루 동안 있었던 일과 감정 상태를 입력하면 Agent가 감정의 원인, 필요한 욕구, 반복되는 생각을 분석하고 회고 질문과 내일의 작은 행동을 제안합니다. 이를 통해 사용자는 감정을 방치하지 않고 기록, 이해, 행동으로 연결할 수 있습니다.
